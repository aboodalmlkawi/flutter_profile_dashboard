# Flutter Personal Profile & Dashboard App

##  App Description
A multi-screen Flutter application designed to showcase a personal profile and an interactive counter. This project demonstrates the use of core Material Components, layout widgets, and state management in Flutter.

##  Features
- **Profile Screen**: Displays user details with a professional layout using Stack, Positioned, and custom Containers.
- **Interactive Gestures**: Includes Double-Tap detection on the profile image and InkWell effects on the profile card.
- **Counter Dashboard**: A dedicated screen to manage and update a counter value using `StatefulWidget`.
- **Navigation**: Smooth transition between the Profile and Counter screens.

##  Widgets Used
This project covers a wide range of Flutter widgets as per the assignment requirements:

| Category | Widgets Used |
|----------|--------------|
| **Core** | `MaterialApp`, `Scaffold`, `AppBar` |
| **Layout** | `Column`, `Row`, `Container`, `Padding`, `Stack`, `Positioned`, `Center`, `SizedBox` |
| **Input & Interaction** | `ElevatedButton`, `IconButton`, `FloatingActionButton`, `InkWell`, `GestureDetector` |
| **Display** | `Text`, `Image.network` (or `Image.asset`), `CircleAvatar`, `Icon`, `SnackBar` |
| **State Management** | `StatelessWidget`, `StatefulWidget`, `setState` |
